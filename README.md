# pierce-cypress-assignment
## Install

1.  Clone the repo
2.  `npm install` or 	`yarn install`


## Run tests
**CLI** - Run CLI

 
```
npx cypress run
```
**Cypress UI** - Run on Cypress UI
 
```
yarn || npm cypress:open
```


## Technology used:

 - Cypress
